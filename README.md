# quiz_app

ダンゴムシクイズとは？
ダンゴムシについてのクイズである。
小学生レベル、中学生レベル、ダンゴムシ博士レベルの３段階の難易度がある。
小学生レベルは問題ページごとに問題を書き込んだので問題の編集が大変だが、
中学生レベル・ダンゴムシ博士レベルは問題ページのテンプレートとすべての問題が集まったページをつくって分離したので問題の編集がしやすい。
点数の管理はproviderをもちいた。
